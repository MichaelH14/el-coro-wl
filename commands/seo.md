# /seo

You have been invoked to produce an SEO analysis and content recommendations.

## Workflow

1. **Current SEO State** — Audit the product's web presence:
   - Crawl or review the product's pages for:
     - Title tags: present, unique, correct length (50-60 chars).
     - Meta descriptions: present, compelling, correct length (150-160 chars).
     - H1 tags: one per page, includes primary keyword.
     - Image alt text: present and descriptive.
     - URL structure: clean, keyword-rich, no unnecessary parameters.
   - Check for technical SEO issues:
     - Page load speed (flags if slow).
     - Mobile responsiveness.
     - Sitemap exists and is valid.
     - Robots.txt is correct.

2. **Keyword Analysis** — Research relevant keywords:
   - Primary keywords: what the product should rank for.
   - Long-tail keywords: lower competition, higher intent.
   - Competitor keywords: what similar products rank for.
   - Keyword gaps: terms competitors rank for but this product doesn't.

3. **Content Suggestions** — Actionable content recommendations:
   - Blog post topics based on keyword gaps.
   - Landing page optimization suggestions.
   - FAQ content based on common search queries.
   - Internal linking opportunities between existing pages.

4. **Meta Description Templates** — Ready-to-use meta descriptions:
   - For the homepage.
   - For key product/feature pages.
   - For blog posts (template with placeholders).

5. **Report** — Structured SEO summary:
   - Current score: qualitative assessment (Good / Needs Work / Poor).
   - Top 5 quick wins: changes that will have the most impact.
   - Content calendar: suggested topics for next 4 weeks.
   - Technical fixes: prioritized list.

## Rules
- Focus on actionable recommendations, not just analysis.
- Quick wins first — prioritize changes with high impact and low effort.
- All meta descriptions must be unique per page — no duplicates.
- Content suggestions should target user intent, not just keywords.
