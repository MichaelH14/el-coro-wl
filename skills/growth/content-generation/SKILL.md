# Content Generation

Generate marketing and product content. Spanish is default language unless explicitly targeting English-speaking markets.

## Preconditions

- Content type identified (social, landing page, announcement, blog)
- Target product known ([Product Name], [Product Name], WhatsApp Bot)
- Key message or feature to communicate is defined
- Target audience segment known

## Steps

### 1. Social Media Posts

Format per platform:
- **WhatsApp status**: 1-2 lines, emoji ok, direct CTA
- **Instagram**: visual-first, caption 2-3 sentences, 5-10 hashtags
- **Twitter/X**: under 280 chars, hook + value + CTA

Content calendar: 3-5 posts per week per active product. Mix:
- 40% value/educational
- 30% product features
- 20% social proof/results
- 10% behind-the-scenes

### 2. Landing Page Copy

Structure:
1. **Hero**: headline (benefit, not feature) + subheadline + CTA
2. **Problem**: what pain the user has (2-3 sentences)
3. **Solution**: how the product solves it (feature highlights)
4. **Social proof**: numbers, testimonials, results
5. **CTA**: single clear action, repeated at bottom

Rules:
- Headline < 10 words
- No jargon -- write at 8th grade reading level
- Benefits over features always
- One CTA per page (not multiple competing actions)

### 3. Feature Announcements

Template:
```
[Product] ahora tiene [feature]!

Que significa para ti:
- [Benefit 1]
- [Benefit 2]

Como usarlo: [1-2 sentence instruction]

[CTA: Try it now / Check it out]
```

Keep under 150 words. Ship announcement same day as feature deploy.

### 4. Blog Posts

Structure:
- **Title**: keyword-optimized, benefit-driven
- **Intro** (100 words): hook + what reader will learn
- **Body** (500-800 words): 3-5 sections with H2 headers
- **Conclusion** (50 words): summary + CTA to product
- Internal links to product pages (min 2 per post)

### 5. Language Rules

- Spanish first for DR/LATAM audience
- Use Dominican colloquialisms sparingly (relatable but professional)
- English only when explicitly targeting US/international market
- Never mix languages in the same piece

## Verification / Exit Criteria

- Content matches the identified type and format requirements
- Spanish is used unless English was explicitly requested
- Single clear CTA exists in every piece
- No technical jargon (readable by non-technical users)
- Content references the correct product features
- Blog posts include internal links to product pages
